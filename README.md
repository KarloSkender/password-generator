# 🔐 Password Generator (Python)

## 📌 Description
This is a beginner Python project that generates secure passwords based on user preferences.

The user can choose:
- Password length (7–16 characters)
- Character types:
  - Lowercase letters
  - Uppercase letters
  - Numbers
  - Symbols

The program ensures:
- At least one character from each selected category
- Randomized final password using shuffle

---

## ⚙️ Features
- Input validation (only numbers allowed)
- User selects character categories (Yes/No)
- Guaranteed inclusion of selected categories
- Random password generation
- Final password shuffle for better security
